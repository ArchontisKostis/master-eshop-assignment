# E-Shop Frontend

A simple React + Vite frontend for the Eâ€‘Shop system, providing customer and store UIs for browsing products, managing carts, and placing orders.

## Features

- **Product browsing**: View and search products with filters
- **Authentication**: Login for customers and stores (JWT-based, backed by the backend service)
- **Shopping cart**: Add, update, and remove items
- **Orders**: Checkout and view previous orders

## Getting Started

### Prerequisites
- Node.js 18+ (recommended)
- npm or yarn

### Install dependencies and configure environment

```bash
cd frontend
cp .env.example .env
npm install
```

### Run in development mode

```bash
npm run dev
```

The app will be available at the URL printed in the terminal (by default `http://localhost:5173`).
